# 🧪 煉金術實驗室：分數平衡大師 (Alchemy Lab: Fraction Balance Master)

這是一個專為 **國小四年級學生** 設計的輕量化、互動式數學網頁遊戲。透過「煉金術」的趣味情境，引導學生直觀地理解分數的大小比較、同分母加減法，以及「單位 1」的完整概念。

---


## 🎮 遊戲內容說明 (Game Stages)

遊戲分為三個階段，對應四年級分數課程的核心指標：

1.  **階段一：重量觀測**
    * *目標*：判斷兩個同分母分數的大小。
2.  **階段二：精準調配**
    * *目標*：計算分數的加減法（需填寫分子與分母）。
3.  **階段三：終極合成**
    * *目標*：理解分數湊成整數「1」的補數概念。

## 🌟 專案亮點 (Highlights)

* **直觀分數呈現**：採用符合教育規範的「垂直分數線」排版，強化學生的視覺數感，而非僅使用 `1/2` 等斜線標示。
* **動態題庫系統**：內建 3階段共 30 題題庫，每次開始遊戲會自動隨機抽取 6 題，確保重複遊玩時的挑戰性。
* **零依賴性 (No Dependencies)**：純原生 HTML/JS/CSS 撰寫，無需安裝 Node.js 或任何框架，下載即可在任何裝置瀏覽器執行。
* **遊戲化反饋**：包含「黃金、銀色、銅色」三種勳章評等，並提供完整的題目回顧與訂正功能。
